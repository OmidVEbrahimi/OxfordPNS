# Oxford Psychological Networks School (OxPNS)

This repository contains the source code for the website of the **Oxford Psychological Networks School (OxPNS)**.

OxPNS is a training programme focused on contemporary network approaches in psychology, medical and social sciences, covering foundational theory and different statistical applications.

## Purpose
The website provides information about the course structure, instructors, schedule, and registration details.
