# OxfordPNS Images

This directory contains images for the Oxford Psychological Networks School (OxPNS) website.

## Required Images

Please add the following images to this directory:

1. **oxford-banner.jpg** - Oxford University banner/hero image for the top of the Home page
2. **network-figure.jpg** - Network figure image for the sidebars on the Home page
3. **course-image.jpg** - Picture for the left side of the Course Details page
4. **registration-image.jpg** - Picture for the left side of the Registration page

The website is designed to gracefully handle missing images by showing placeholder elements.
